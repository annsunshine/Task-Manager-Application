The application contains main window which displays buttons called "Add new task", "Edit existing task" and "Check created tasks".
<img width="1596" height="1340" alt="image" src="https://github.com/user-attachments/assets/6e71a490-a53e-4a1a-a8ff-718c465d335c" />
If the user wants to create a new task, them should click on "Add new task" botton. Thanks to that, a new window with Task's number, name, deadline, priority and status entries will open.
Worth mentioning is fact that the user has to double click on the deadline list to choose a date. However, for selecting priority and status, only one click is necessary. At the end the user should click on "Save task" button to save the task.
<img width="1588" height="744" alt="image" src="https://github.com/user-attachments/assets/89a4c23f-8b6e-4a11-bdd5-de27e95099b3" />
In a situation when the user wants to edit already saved task, them ought to click on "Edit existing task" button. It will enable opening a Edit task window, with Task's number, name, deadline, priority and status entries.
Here the user can change any item connected to a particilar task and save it by clicking on "Update task" button.
<img width="1660" height="736" alt="image" src="https://github.com/user-attachments/assets/8bb70004-c216-434e-9104-903f837cf3e2" />
However, if the user wants to check all tasks them introduced to the application, "Check created tasks" button has to be clicked. Thanks to that a new window will open and display all activities.
